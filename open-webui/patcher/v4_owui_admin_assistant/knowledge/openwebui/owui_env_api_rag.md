# Open WebUI Environment, API, and RAG

## Environment Variables
- DATA_DIR
- WEBUI_SECRET_KEY
- DATABASE_URL
- CORS_ALLOW_ORIGIN

## API Patterns
- Auth: signup/signin
- Models: create/update/get
- Knowledge: create/upload/attach

## RAG Setup
- Create KB
- Upload docs
- Attach KB to model
- Test retrieval with known fact queries

## Workspace Practices
- Separate dev/test/prod data directories.
- Keep backups before patching/upgrades.
- Track model and prompt revisions.
