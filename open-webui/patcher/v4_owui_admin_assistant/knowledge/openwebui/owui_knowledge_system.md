# Open WebUI Knowledge System

## Purpose
Support high-quality retrieval over curated domain documents.

## RAG-Friendly Authoring Rules
- One topic per document.
- Stable headings and section names.
- Include examples and anti-patterns.
- Keep terminology consistent.

## Ingestion Practices
- Upload Markdown docs grouped by domain.
- Re-index after updates.
- Avoid duplicate documents with overlapping titles.

## Quality Checks
- Retrieval precision by query tests.
- Missing-section lint checks.
- Best-practice checklist coverage.
